Step 1: Upload seed file

Copy your transformedData.parquet into the shared_data Docker volume as seed.parquet:

docker cp transformedData.parquet $(docker ps -qf name=web):/data/seed.parquet

Step 2: Generate 2-year dataset

Replicates the 15-day seed 48 times (~1.4B rows). Takes 45-90 min on N100.

Step 3: Load into databases

Loads data into all databases. Takes 4-8 hours on N100.

Step 3b: Generate & load hierarchy

Creates a synthetic EAN\u2192supplier/category/sub-category mapping table. Run once after data is loaded.
